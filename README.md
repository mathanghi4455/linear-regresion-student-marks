# linear-regresion-student-marks
# Student Score Prediction

This is a simple Machine Learning project using Linear Regression.

## Features

* Predicts marks based on study hours
* Uses sklearn library
* Simple dataset

## How to Run

1. Install dependencies:
   pip install numpy scikit-learn matplotlib
2. Run:
   python3 linear_regression.py

## Output

Predicts student marks based on input hours.
